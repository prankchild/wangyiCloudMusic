



# 仿网易云音乐小程序项目

实现功能：调用接口完成登录功能，首页轮播图，每日推荐歌曲，排行榜，视频界面，个人界面，最近播放，搜索歌曲，热搜榜等功能

调用的接口为: https://github.com/w4ctech/NeteaseCloudMusicApi
